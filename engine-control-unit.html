Engine Control Unit — Karman Space Programme

STM32-based ECU for a liquid bipropellant rocket engine.

This project involved the design and implementation of an Engine Control Unit for a liquid bipropellant rocket engine.

Key Features

  • STM32 microcontroller
  • 1 Mbps CAN bus
  • 24V, 2A solenoid valve drivers
  • High-speed SPI sensors

My Contributions

Firmware development, CAN communication, and system-level debugging.
